Please Read: BGG Import Instructions


To import your collection from BoardGameGeek you need to activate the unique API url
for your collection. To do this visit the following URL after replacing "USERNAME"
with your BGG username:

http://www.boardgamegeek.com/xmlapi/collection/USERNAME?own=1

You should see 'Your request for this collection has been accepted and will be
processed.' After several minutes refresh the page and you should see the XML output
of your collection. Then you can proceed with the import below

Note: The import will take 1 second for every game in your collection.
